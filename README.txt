This is The End Update mod for Minecraft :D


Additions:
-Limestone
-Limestone Bricks
-Purple Forest, Vulcanic End Biomes and Purpur Fields (before public beta 3 this biomes can generate in overworld)
-New End Dimension (portal must be created from endstone)
-End Magma Block
-End Purple Rock
-End Vulcanic Stone
-End Purple Tree Stem
-End Vulcanic Tree Stem
-End Purple Tree Leaves
-End Vulcanic Tree Leaves
-End Flyblock
-Purpur Ore
-Hardened Obsidian
-Brimstone
-Cheese
-Wither Powder
-Purple and Vulcanic planks
-End Liquid
-Purpur Sprouts
-Purpur Berries
-Mob Repellant
-aww man... (It's a joke item (I like 20w14infinite))

-GIMP 2.10 To create textures
-MCreator 2019.5 to compile mod
